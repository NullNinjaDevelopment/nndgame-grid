# nndgame-grid

# Usage

### Running tests
Run any of the following commands to execute tests
* `npx jest`
* `npm test`
* `npm t`

### Run tests on every file change
Run any of the following commands to start a process that executes tests every time a file changes
* `npx jest --watch`
* `npm test -- --watch`
* `npm t -- --watch`
